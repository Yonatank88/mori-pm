# PM Foundations

A product-management basics program and its working artifacts.

**Site:** https://yonatank88.github.io/mori-pm/

- **Course** (`index.html`) — the program structure and weekly units, rendered live from [`PM Foundations.md`](PM%20Foundations.md).
- **Artifacts** (`artifacts.html`) — per-program working documents under [`Programs/`](Programs/).

The pages fetch the markdown files at load time, so editing any `.md` file and pushing updates the site automatically. When adding a new document or folder, also add its path to the `TREE` list in `artifacts.html`.
